Detailed Description

template<class T>
class mathx::array< T >

This class is a variable size, random-access data structure for type T.

As working with raw pointers can be dangerous and hard to debug, this class serves as a wrapper of a pointer container of type T. Included in this class are helper methods to accomplish tasks such as adding to vectors, doting two vectors, and dynamically extending the container via a push method.

Examples:
array.cpp, interpolation.cpp, linsolv.cpp, and vectors.cpp.
